# Temp Email – rungrot.com

เว็บ Gen Email + กล่องจดหมาย สำหรับโดเมน rungrot.com (Node.js server)

## โครงสร้าง

- `public/` – ไฟล์เว็บ (index.html, style.css, app.js)
- `server.js` – เซิร์ฟเวอร์ Express + รับเมล SMTP + API Inbox
- `package.json` – dependencies

## ติดตั้งและรัน

```bash
cd C:\Users\dev\Desktop\Email
npm install
npm start
```

- เปิดเว็บ: **http://localhost:3000**
- SMTP รับเมล: **พอร์ต 2525** (ค่าเริ่มต้น)

## ตัวแปรสภาพแวดล้อม (ถ้าต้องการ)

- `PORT` – พอร์ต HTTP (ค่าเริ่มต้น 3000)
- `SMTP_PORT` – พอร์ตรับเมล SMTP (ค่าเริ่มต้น 2525, ใช้ 25 ตอน deploy จริงถ้าเซิร์ฟเวอร์อนุญาต)
- `DOMAIN` – โดเมนอีเมล (ค่าเริ่มต้น rungrot.com)

## การตั้ง MX record

ให้เมลส่งมาที่เซิร์ฟเวอร์นี้ได้ ต้องตั้งที่ผู้ให้บริการโดเมน (rungrot.com):

1. สร้าง **MX record** ของโดเมน `rungrot.com`
2. ชี้ไปที่ **ชื่อโฮสต์หรือ IP ของเซิร์ฟเวอร์ที่รัน Node.js นี้**
3. **Priority** ใช้ค่าเช่น 10
4. พอร์ตรับเมลมาตรฐานคือ **25** – ถ้าใช้พอร์ตอื่น (เช่น 2525) บางผู้ให้บริการโดเมน/โฮสต์อาจต้องตั้งส่งเมลมาพอร์ตนั้น (หรือใช้ relay)

หมายเหตุ: บน VPS/เซิร์ฟเวอร์จริง มักต้องรันด้วยพอร์ต 25 เพื่อให้เมลจากอินเทอร์เน็ตส่งเข้ามาได้ ถ้ารันบน Windows/local ใช้ 2525 สำหรับทดสอบก่อนได้
